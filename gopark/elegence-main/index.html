About GoPAK

what is GoPAK

GoPAK은 국내의 명소들을 방문하면 발급받을수 있는 블록체인 코인입니다.국내여행활성화 를 장려하고 결과적으로 내수경제활성화 를 목표로 하고있습니다.

how to use gopak

MARKET

여행을 통해 얻은 코인으로 또다른 여행을 즐기거나 온 오프라인에서 각 지역의 특산물을 구매할수있습니다.

RANK

획득한 코인은 블록체인으로 기록되어 온라인에서 랭킹을 확인할수있습니다.

Memory

나의 여행기록이 체인으로 연결되어 죽어서도 기억됩니다.
